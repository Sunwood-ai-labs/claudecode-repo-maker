# 🎲 Dice App

シンプルで使いやすいサイコロアプリケーション

## 📝 概要

Dice Appは、ブラウザ上で動作するシンプルなサイコロアプリケーションです。クリック一つでサイコロを振ることができ、ボードゲームやテーブルトークRPG、意思決定など様々な場面で活用できます。

## ✨ 機能

- 🎲 1〜6の目をランダムに表示
- 🖱️ ワンクリックでサイコロを振る
- 🎨 視覚的に分かりやすいUI
- 📱 レスポンシブデザイン対応
- ⚡ 高速・軽量

## 🛠️ 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 インストール方法

### リポジトリのクローン

```bash
git clone https://github.com/Sunwood-ai-labs/dice-app.git
cd dice-app
```

### 使用方法

1. `index.html` をブラウザで開く
2. 「サイコロを振る」ボタンをクリック
3. ランダムな結果が表示される

## 💻 開発

### ローカル開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/Sunwood-ai-labs/dice-app.git
cd dice-app

# ローカルサーバーを起動（任意）
# Python 3の場合
python -m http.server 8000

# Python 2の場合
python -m SimpleHTTPServer 8000
```

ブラウザで `http://localhost:8000` にアクセス

### プロジェクト構成

```
dice-app/
├── index.html      # メインHTMLファイル
├── style.css       # スタイルシート
├── script.js       # JavaScript ロジック
└── README.md       # このファイル
```
